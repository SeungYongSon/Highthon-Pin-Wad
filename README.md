# Highthon-Pin-Wad
## 하이톤 2회 졸업을 주제로 만들어진 앱입니다.  

자신이 활동하는 장소나 만나고 싶은 장소를 지도에 핀을 찍은 다음에  
동창들이 핀들의 정보를 보고 찾아가서 친구들을 만나고 서로 놀고 이야기를 나누겠끔하는 앱입니다.
